# Phishing Website Detection

## Overview
This project focuses on detecting phishing websites using machine learning and deep learning techniques. Phishing attacks attempt to trick users into revealing sensitive information by mimicking legitimate websites. This system analyzes visual and structural features of websites to classify them as phishing or legitimate.

## Objectives
- Build a machine learning pipeline for phishing detection
- Extract meaningful features from website images
- Train and evaluate classification models
- Compare model performance using standard evaluation metrics

## Project Pipeline
1. Data Collection
2. Data Preprocessing
3. Feature Extraction
4. Dataset Splitting (Train/Test)
5. Model Training
6. Model Evaluation
7. Model Saving

## Technologies Used
- Python
- Scikit-learn
- PyTorch
- TensorFlow
- OpenCV
- XGBoost
- LightGBM

## Evaluation Metrics
The models are evaluated using:
- Accuracy
- Precision
- Recall
- F1 Score

## Applications
- Cybersecurity systems
- Browser phishing detection
- Online fraud prevention
- Website authenticity verification
